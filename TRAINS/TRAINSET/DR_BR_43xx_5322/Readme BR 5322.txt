BR 43xx Class 5322 

3D Model built from drawings and textured by Dave Robinson
Sounds By Stuart Williamson File ID:6481  
Cabview By Lew price 
Eng file by Dave Robinson

-----------------------------------------------------------------------------
The 3D model has been built using TrainSim Modeller

This is a model of BR 43xx Class 5322 in British Railways Plain Black livery

This loco comes with changeable headlamps and disk, simple copy over all the dead lamps/disks first from the dead lamp/disks 
folder in to the main folder and then copy from the live lamps/disks folder the lamps or disks you require on the loco to the main folder. 

To install copy DR_BR_43xx_5322,common.cab folders to your trainset which can be found here 
(C:\Program Files\Microsoft Games\Train Simulator\TRAINS\Trainset)
you will also need to have Stuart Williamson GWR Sound set file ID 6481

Any suggestions of help or constructive comments welcome.
