Brighton H2 Atlantic Class BR 32424 4-4-2 steam locomotive

3D Model built from drawings and textured by Dave Robinson
Sounds By Stuart Williamson File ID:2910 
Cabview By DarwinS (Darwin Smith)File ID:29169 

 
-----------------------------------------------------------------------------
The 3D model has been built using TrainSim Modeller

This is a model of a Brighton H2 Atlantic Class BR 32424 Beachy Head
In BR Lined Black Livery 

This loco comes with changeable headlamps simple copy over all the dead lamps or first from the dead lamp 
folder in to the main folder and then copy from the live lamps folder the lamps you require on the loco to the 
main folder.

Any suggestions of help or constructive comments welcome.

To install simpley copy the 3 files ( DR_BR_H2_32424,common.sound,common.cab ) to your Trainset folder which can be found here
C:\Program Files\Microsoft Games\Train Simulator\TRAINS\Trainset 
You will also need UKTS 2910 2cSndSetV1-1 by Stuart Williamson File ID:2910 and LMS (ex CR) class 439 cab view by DarwinS (Darwin Smith) File ID 29169
you will then need to make a consists using activtie editor in Train Simulator Editors & Tools
