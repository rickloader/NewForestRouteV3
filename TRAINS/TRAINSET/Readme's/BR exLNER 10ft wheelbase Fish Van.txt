HISTORY
~~~~~~~
This was the second design of LNER fish van. It was based on the current standard goods van. The first batch had a 9ft wheelbase. This was followed by a 10ft version and finally and longer 12ft WB version.

INSTALLATION
~~~~~~~~~~~~

Unzip to tenporary folder and copy, move or drag to your Trainsets folder.
This version will overwrite any previous version.

You are now ready to create a consist containing this model by using the Activity Editor
supplied with Microsoft Train Simulator - the model will not be available in the simulator unless you do this.

WARNING: It is not recommended that you delete these files once installed unless you remove all consists and activities which use it.
